# Notes Service - Project Structure

This document outlines the folder structure and organization of the Notes Service project.

## Directory Structure

```
notes-service/
├── .github/
│   └── instructions/          # Project instructions and guidelines
├── Docs/                       # Documentation
│   └── project-structure.md   # This file - project structure reference
├── src/                        # Source code
│   ├── Notes.Api/             # Backend API (.NET)
│   └── notes-ui/              # Frontend UI
└── README.md                   # Project overview
```

## Directory Descriptions

### `.github/instructions/`
Contains project-specific instructions, guidelines, and workflow documentation.

### `Docs/`
Central location for all project documentation including architecture, API specs, and guides.

### `src/`
Root directory for all source code.

### `src/Notes.Api/`
Backend API implementation using .NET. This will contain:
- Controllers
- Services
- Models
- Database configurations
- API endpoints

### `src/notes-ui/`
Frontend user interface. This will contain:
- UI components
- Views
- Client-side logic
- Styling

## Getting Started

Refer to the individual README files in each directory for specific setup and development instructions.
